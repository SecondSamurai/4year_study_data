                       Практическая работа N 5
                   Обработка многомерных массивов          

   Требования к выполнению практической работы:
1. В тетради должен быть записан текст задания.
2. Во время работы программы на экране должны быть:
   - номер и тема практической работы;
   - ФИ и группа студента;
   - условие задачи.
3. Необходимо выполнить полный контроль входных данных при помощи цикла.
4. Ответ необходимо дать полный по формату 2 знака после запятой. Причем 
   матрица должна быть напечатана в виде таблицы, а одномерный массив
   (если он в программе необходим) в строку.
5. Требуется использовать в программе команды работы с цветом и курсором,
   работы с окнами.
6. Текст программы должен быть записан со всеми необходимыми отступами.
7. Размер и элементы матрицы должны вводится с клавиатуры.
8. Файл должен называться   pract8

   Задания:

Вариант 1

	Просуммировать элементы строк матрицы. Результат получить в одно-
мерном массиве. Матрицу ввести с клавиатуры.


Вариант 2

	Просуммировать элементы столбцов матрицы. Результат получить в 
одномерном массиве. Матрицу ввести с клавиатуры.


Вариант 3

	Для заданной квадратной матрицы сформировать одномерный массив из 
ее диагональных элементов. Найти след матрицы, суммируя элементы одномерного 
массива. Матрицу ввести с клавиатуры.


Вариант 4

	Заданы матрица и вектор. Получить их произведение. Матрицу и вектор 
ввести с клавиатуры.


Вариант 5

	Для каждой строки матрицы найти максимальный элемент и записать их 
в одномерный массив. Матрицу ввести с клавиатуры.


Вариант 6

	Для каждой строки матрицы найти минимальный элемент и записать их 
в одномерный массив. Матрицу ввести с клавиатуры.
 

Вариант 7

	Просуммировать две матрицы. Матрицы ввести с клавиатуры.


Вариант 8

	Поменять местами L-ую и К-ую строки матрицы. Матрицу ввести с 
клавиатуры.


Вариант 9

	Поменять местами L-ый и К-ый столбцы матрицы. Матрицу ввести 
с клавиатуры.


Вариант 10

	Транспонировать матрицу, то есть заменить строки матрицы столбцами. 
Матрицу ввести с клавиатуры.


Вариант 11

	Для каждого столбца матрицы сосчитать количество положительных
элементов. Результат сохранить в одномерном массиве. Матрицу ввести с кла-
виатуры.


Вариант 12

	Для каждого столбца матрицы сосчитать сумму четных элементов. 
Результат сохранить в одномерном массиве. Матрицу ввести с клавиатуры.


Вариант 13

	Для каждого столбца матрицы найти среднее арифметическое. Резуль-
тат поместить в одномерный массив. Матрицу ввести с клавиатуры.


Вариант 14

	Для каждой строки матрицы найти сумму отрицательных элементов. 
Результат поместить в одномерный массив. Матрицу ввести с клавиатуры.


Вариант 15

	Для каждой строки матрицы найти среднее геометрическое. Результат 
поместить в одномерный массив. Матрицу ввести с клавиатуры.


Вариант 16

	Для каждого столбца матрицы найти произведение элментов, кратных 3. 
Результат поместить в одномерный массив. Матрицу ввести с клавиатуры.


Вариант 17

	Для каждой строки матрицы найти количество элементов, больших L. 
Результат поместить в одномерный массив. L, матрицу ввести с клавиатуры.


Вариант 18

	Для каждого столбца матрицы найти количество элементов, кратных 5. 
Результат поместить в одномерный массив. Матрицу ввести с клавиатуры.
